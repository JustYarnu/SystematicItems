An every evolving ecosystem of items that evolve over time. 
- Every item contains attributes that can be mutated according to a massive set of rules (let's say 100) 
- If an item evolves it loses health, eventually being inert and getting archived (you view it). The items that aren't archieved you can also view. 
- Rules are kind of like factions, they have their own weight, some factions reduce weights of others and can change relation (enemy, friend, changing their weight gain/loss for other factions). 
- Minor chance for items to fail mutating -> item gets converted into currency Currency is used by users to increase faction weight temporarily (idk how you'd do this with just github though) Everything will be stored in a bit database you can search items for and query.
- Eventually everything is reset and archives, should leave a little bit of influence (cycles)
